# 小米澎湃OS 钱包XPosed模块

## 简介
本仓库提供了一个小米EU澎湃OS系统的钱包XPosed模块，刷入后可以实现公交、地铁、门禁等功能的支持。该模块基于小米12S制作，理论上适用于其他型号的小米设备。

## 功能特点
- **公交支持**：刷入模块后，可以使用小米钱包进行公交卡的刷卡操作。
- **地铁支持**：支持地铁卡的刷卡功能，方便快捷。
- **门禁支持**：通过小米钱包实现门禁卡的功能，无需携带实体卡片。

## 适用系统
- MIUI 14
- 澎湃OS 1
